# NUVU — Unveil the New You 🧬

![NUVU Logo](https://raw.githubusercontent.com/placeholder/nuvu-logo.png) (Replace with actual logo if available)

**NUVU** is a premium, high-conversion web platform for bio-optimization and nutritional supplementation. Inspired by the surgical precision of modern medical brands like Medvi, NUVU offers a sophisticated user experience focused on health, longevity, and performance.

## ✨ Key Features

- **EU Regulatory Compliance**: Fully optimized for European health regulations (Reg. CE 1924/2006, 432/2012). No illegal medical claims — just pure, evidence-based nutritional science.
- **Multilingual Support**: Integrated dynamic translation (ES, EN, FR, DE, LB) using a seamless custom implementation.
- **Precision Assessment Tool**: A 4-phase clinical-grade questionnaire that guides users to their ideal supplement selection based on biometrics and lifestyle.
- **AI VIP Assistant**: Integrated chatbot powered by advanced logic to provide real-time guidance and support.
- **Premium Aesthetics**: Obsidian Black & Surgical Gold design system with high-end glassmorphism and smooth animations.
- **Responsive & Minimalist**: Optimized for all devices with a focus on core conversion elements.

## 📂 Project Structure

- `index.html`: The main landing page featuring the brand story and product showcase.
- `assessment.html`: The 4-phase interactive health questionnaire.
- `product-*.html`: Detailed product pages for Thermo Matrix, Neuro Focus, and Premium Whey.
- `styles.css` & `product.css`: The core design system and component styling.
- `translate.js`: Custom seamless translation logic.
- `chatbot.js`: The "AI Conserje" logic.
- `main.js`: Core UI interactions and navigation.

## 🚀 Getting Started

Simply open `index.html` in any modern browser to experience the NUVU brand.

```bash
# Clone the repository
git clone https://github.com/yourusername/nuvu-web.git

# Navigate to the folder
cd nuvu-web

# Open index.html
open index.html
```
